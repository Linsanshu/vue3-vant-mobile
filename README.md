<h1 align="center">Fooyo-Vue3-Mobile</h1>

<p align="center">Mobile Template of FooyoTech, built on Vue3、Vite3、Vant4、Pinia2、Vitest、TypeScript </p>

## Install and use

- Installation dependencies

```bash
cd vue3-vant-mobile

# with yarn
yarn install

```

- run

```bash
yarn dev
```

- build

```bash
yarn build
```
